Об авторе

Зовут меня Белоусов Дмитрий. В рамках учебного курса были изучены следующие методы и технологии: основы верстки, понятие семантики и правильности выбора тегов, понятие классов и наименование в соответствии с БЭМ, основы CSS, адаптивная верстка, Pixel Perfect, технологии Grid и Flex, шрифты и возможности при их подключении, изучены основы Javascript, получено понятие о примитивах, объектах, работа с ними, функции, области видимости функций. Большое внимание уделено конструкции с new, а также методам функций, понятию this. Классы и работа с ними. Работа с API, fetch запросы, понятие промиссов и работа с ними, инфраструктура веб-приложений: работа с git, использование GitHub как удаленного репозитория, понятие модулей в JS, работа с Webpack и сборка готового проекта. Выполненные проекты (прошли код-ревью у наставников): - Статический сайт. Адаптивная верстка. Использованы в работе HTML и CSS. - Динамическое наполнение сайта в зависимости от API запроса, работа с проммисами, валидация формы. Формирование интерактивной ленты с карточками изображений. С возможностью ставить лайки, удалять, добавлять карточки. Использованы HTML, CSS, Javascript. -Аналитический сайт формирующий по API запросу список карточек с новостями, формирование диаграммы частоты запроса за последнюю неделю, формирование карусели последних коммитов на портале GitHub.com.Использованы HTML, CSS, Javascript, Node.js, Webpack. В принципе, вы сейчас на нем и находитесь.

Все работы можно найти в моем рапозитории на сайте GitHub.com по ссылке расоположенной внизу страницы.
